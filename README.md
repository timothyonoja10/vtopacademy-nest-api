## Description

A nest js backend api for accessing the following resources: schools, exams, classes, subjects, topics, tutorial videos, and practice questions

## Installation

```bash
$ npm install
```

## Environment variables in .env or .development.env

```bash
PORT=  

DATABASE_HOST=
DATABASE_PORT=
DATABASE_USERNAME=
DATABASE_PASSWORD=
DATABASE_NAME=

MONGODB_URI=

THROTTLE_TTL=60000n 
THROTTLE_LIMIT=1000

JWT_SECRET=tuirnoo22
JWT_EXPIRES_IN=86400s 

EMAIL_SERVICE_HOST=
SERVICE_PORT=
EMAIL_SERVICE_USERNAME=
EMAIL_SERVICE_PASSWORD=
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
